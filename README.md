# protosignet

Designing network motifs via ODE modeling and NSGA-II.

## Getting Started

### 1. Install [UV](https://docs.astral.sh/uv/getting-started/installation/)

```bash
$ curl -LsSf https://astral.sh/uv/install.sh | sh
```

### 2. Edit Objective Function

protosignet/main.py

### 3. Run Code

```bash
$ uv run main
```
